Semantic Colors

Semantic colors include various colors assigned to meaningful contextual values that are saturated enough to meet contrast requirements when using white or light font colors, these values cascade to buttons, alert, badges, etc.

  • @brand-primary: #000
  • @brand-primary-accent: @wtwViolet
  • @brand-success: #5cb85c;
  • @brand-info: #5bc0de;
  • @brand-warning: #f0ad4e;
  • @brand-danger: #d9534f;

Grayscale Colors

Grayscale colors provide quick access to commonly used shades of black.

  • @gray-base: #000;
  • @gray-darkest: lighten(@gray-base, 26.5%); // #444
  • @gray-darker: lighten(@gray-base, 40%); // #666
  • @gray-dark: lighten(@gray-base, 53.5%); // #888
  • @gray: lighten(@gray-base, 66.5%); // #aaa
  • @gray-light: lighten(@gray-base, 86.5%); // #ddd
  • @gray-lighter: lighten(@gray-base, 93.5%); // #eee
  • @gray-lightest: lighten(@gray-base, 96.7%); // #f7f7f7
